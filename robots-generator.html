How to Create a Robots.txt File

                

                
    
                    
  1. Configure the User-agent settings. Leave this as * if you want your robots.txt rules to apply to all search engines, or set it to only apply to specific crawlers.
    2. 
                    
    
                    
  2. Add directories to your Disallow list. These are directories that will be marked as NOT intended for indexing by crawlers.
    3. 
                    
    
                    
  3. Add directories to your Allow list. These are directories that will be marked as intended for indexing by crawlers.
    4. 
                    
    
                    
  4. Optionally, use the Bulk Paste options for Allow and Disallow.
    5. 
                    
    
                    
  5. Click the green Generate robots.txt button.
    6. 
                    
    
                    
  6. Either copy the generated text directly from the output area below the button and paste it into a text editor, or press the Download robots.txt button to download the file.
    7. 
                    
    For more assistance, refer to our guides, tutorials, and troubleshooting page.
